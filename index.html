Current Project

As a Jira Admin and Developer, I got directly engaged in the migration process of a Jira Data Center from one cloud provider to another. While migrating from AWS to OCI, I gained knowledge in numerous areas, such as Terraform, OCI CLI and SDK. With those technologies, an autoscaling group was created, where a custom image is used to provision as many nodes as required based on specific metrics. Another benefit of this configuration is the simplified upgrade process in the future. I led the process of performing load tests and benchmarks in order to determine the best number of resources allocated (CPU, RAM, Database, NFS). As a last step, with numerous integrations already in place, I had to make sure that the networking side was correctly set up, before the migration started.
